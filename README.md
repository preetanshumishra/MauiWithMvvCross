# MauiWithMvvCross

A .NET MAUI sample demonstrating MVVM using the MvvmCross framework, the enterprise-grade cross-platform MVVM solution.

## Overview

This project showcases MvvmCross integration with .NET MAUI for robust MVVM implementation:
- **MvvmCross Framework** - Mature, production-ready MVVM framework with extensive features
- **IoC Container** - Built-in dependency injection and service resolution
- **ViewModel-First Navigation** - Navigate by routing to view models instead of views
- **Data Binding** - Powerful data binding with converters and validation

## Tech Stack

- .NET 10.0
- .NET MAUI 10.0.10
- MvvmCross 9.1.1
- Community Toolkit MVVM 8.4.0

## Quick Start

```bash
# Build the project
dotnet build

# Run on iOS
dotnet run -f net10.0-ios

# Run on Android
dotnet run -f net10.0-android
```

## Key Features

- MvvmCross IoC container setup and configuration
- ViewModel-first navigation pattern
- Presentation attributes for routing configuration
- Custom value converters
- Cross-platform view model sharing
- Plugin system support
